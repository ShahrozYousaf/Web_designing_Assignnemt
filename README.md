# Registration Form UI

A modern and responsive Registration Form built using pure HTML and CSS. This project demonstrates form design, responsive layouts, custom styling, and user-friendly input fields.

## 📌 Overview

This project features a clean registration interface with a dark-themed card design and modern purple accent colors. It includes all essential fields required for user registration.

## ✨ Features

* Responsive registration form
* Modern dark UI design
* Full Name, Parent Names, and Date of Birth fields
* Gender selection using radio buttons
* Email and Phone inputs
* State and City fields
* Address textarea
* Password and Confirm Password fields
* Register and Sign In buttons
* Hover and focus effects
* Mobile-friendly layout

## 🛠️ Technologies Used

* HTML5
* CSS3
